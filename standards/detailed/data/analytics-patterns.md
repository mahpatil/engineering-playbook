# Analytics Patterns

Common analytics query patterns for reporting, dashboards, and business intelligence use cases.

---

## 1. Analytics Patterns

| Pattern | Description | Example |
|---------|-------------|---------|
| Rolling aggregation | Last N periods | 7-day moving average |
| Cumulative | Running total | YTD revenue |
| Funnel analysis | Conversion steps | Signup → Activation → Paid |
| Cohort analysis | Group by time | Monthly retention by signup cohort |

---

## 2. When to Use

- **Rolling aggregation**: Trend monitoring, anomaly detection
- **Cumulative**: Financial reporting, progress tracking
- **Funnel analysis**: Product onboarding, conversion optimization
- **Cohort analysis**: Retention analysis, lifetime value
